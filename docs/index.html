Features

  • Trading Simulator: $1 → $1B in <60s with autonomous algorithm
  • Prediction Markets: Real-time Polymarket feeds with probability filters
  • Monte Carlo: 5K-path simulations with bull/base/bear scenarios
  • Live Data: Yahoo Finance stocks, commodities, crypto via CoinGecko
  • Vercel Blob Cache: Daily updates at 8 AM UTC for instant load

Quick Start

git clone https://github.com/nulljosh/rise.git
cd rise
npm install
npm run dev

Tech Stack

  • React 19 + Vite
  • Recharts for visualization
  • Polymarket + Yahoo Finance APIs
  • Custom Monte Carlo engine
  • Vercel deployment with Blob storage

Philosophy

The 10MB Dashboard - Every feature must justify its bytes. Speed is sacred, latency kills alpha.
